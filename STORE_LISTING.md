# Chrome Web Store Listing — PropSight

## Short Description (132 chars max)

```
Reveals hidden rental constraints on PropertyGuru SG — cooking, landlord, address registration — as color-coded tags on every listing.
```

## Detailed Description

```
PropSight is a free, privacy-first Chrome extension for renters in Singapore. It scans PropertyGuru rental listings and instantly reveals critical information that's usually buried in long descriptions — displayed as color-coded tags right on the search results page.

🔍 WHAT IT DOES

PropSight automatically reads listing descriptions and extracts key attributes:

• Cooking Policy — Heavy cooking OK, light cooking only, or no cooking
• Owner Occupancy — Landlord lives in, or owner not staying
• Address Registration — Allowed or not allowed (critical for work pass holders)
• Room Type — Master room, common room, studio, whole unit, or partition
• Bathroom — Private or shared
• Agent Fee — No agent fee, half month, or has fee
• Gender Preference — Male only, female only, couples OK, or any
• Visitor Policy — Allowed, restricted, or not allowed
• Lease Term — Minimum months extracted from description
• Amenities — Aircon, WiFi, furnished, near MRT, and more

🏷️ COLOR-CODED TAGS

Tags appear directly on listing cards so you can scan results at a glance:
• 🔴 Red = Deal-breakers (no cooking, landlord lives in, no address reg)
• 🟢 Green = Positive signals (cooking OK, no landlord, address reg allowed)
• 🟡 Yellow = Caution (light cooking only, visitor restrictions)
• 🔵 Blue = Neutral info (room type, lease term, amenities)

🔧 FILTER PANEL

A floating filter panel on search results pages lets you:
• Filter by any extracted tag dimension (cooking, owner, bathroom, etc.)
• Toggle "Hide unanalyzed" to show only listings with confirmed tags
• See a live match counter (e.g., "Showing 5 / 42 listings")
• Filters persist across page navigations

🔒 PRIVACY

• No backend server — everything runs in your browser
• No API keys or accounts required
• No data sent anywhere — tags are cached locally for 24 hours
• Open source: https://github.com/niclasliu/PropSight

💡 WHO IS THIS FOR

• Expats who need to verify address registration eligibility
• Tenants who need cooking-allowed units
• Anyone tired of clicking into every listing to check basic constraints

Works on PropertyGuru Singapore rental search results, room rental pages, map view, and individual listing pages.
```
